Windows 95

Windows 95 has been the well-known operating system launched in 1995. Some major features introduced to this operating system include a redesigned shell interface, the first Start menu, Plug-And-Play support, and "The Microsoft Sound" as a startup sound, created by Brian Eno.

Windows 95 was sold alongside Microsoft Plus! for Windows 95, as shown in a picture below. Microsoft Plus! for Windows 95 came with Internet Explorer version 1.0, as well as its exclusive themes. Microsoft Plus! adds an enhancement to Windows that excites the fans around the world.

Windows 95 at the Microsoft Visitor Center in Redmond, Washington.

Windows NT Workstation Version 4.0

Designed for business users, Windows NT Workstation Version 4.0 is suitable for intensive computing tasks and servers. It comes with the Windows 95 interface integrated into the Windows NT architecture, making this operating system more stable and easy to use. Windows NT 4.0 comes in multiple editions: Workstation, Server, and Embedded. It supports Alpha, MIPS, Intel, or PowerPC CPUs.

Windows 98

Released in 1998, Windows 98 is an improved version of the Windows 9x family. It comes with new features such as an improved user interface, shell enhancements, web integrations, and a robust USB support. Windows 98 can also support TV tuners through WebTV. Windows 98 Second Edition was released one year later, in 1999; it comes with improved USB support, improved audio and modem support, and an introduction to Internet Explorer 5.

Windows 98 also comes with the dedicated ACPI support which allows your computer to either hibernate or go into the stand-by mode, as well as the multi-monitor support, which uses up to 9 monitors from one PC.

Windows 2000 Professional

Built on Windows NT Technology, Windows 2000 Professional claims to be one of the most successful operating systems to date. It comes with an improved Plug-And-Play feature with full ACPI and Windows Driver Model support, usability enhancements, and mulit-language support. It comes with 3 editions: Professional, Server Family, and Advanced Server.

Windows Millennium Edition

Proven to be the last Windows 9x version, Windows Millennium Edition is suitable for home computing needs. It comes with improved boot times, Universal Plug-And-Play, Internet-based games, System Restore (which restores to the previous system from what was recently changed), and Automatic Updates.

Multimedia Features!

Windows XP

Windows XP is the most popular operating system created by Microsoft. Launched on October 25th, 2001, XP unites both business and home users thanks to the power of Windows NT technology. It provides Wi-Fi and Bluetooth support for better connectivity. The Start menu has been revamped, now showing frequently used applications, recently opened documents, and so much more. The "Switch Users" feature allows computer users to switch to the other user without having to close all applications and log out.

Windows XP comes with two client editions:

Windows XP Home Edition
  • Designed for home computing needs, built on the Windows NT Technology. Home Edition features enhanced features, such as:
    • Windows Messenger, for instant messaging services
    • Windows Media Player 8
      • Can be upgraded up to Windows Media Player version 11
    • Internet Explorer 6
      • Can be upgraded up to Internet Explorer 8 (requires Service Pack 2 or up)
    • Windows Movie Maker
      • Edits videos with music and titles
Windows XP Professional
  • Designed for business and power users, Windows XP Professional comes with all features found in Home Edition, plus:
    • The ability to use remote desktop to connect to another Windows XP computer remotely
    • Offline Files and Folders
    • Group Policies
    • Encrypting File System
    • Multilingual Support

Additionally, Windows XP comes in several editions:

Windows XP Media Center Edition

The best Windows XP edition for home entertainment personal computers. Windows XP Media Center Edition comes with Windows Media Center, the first one-of-a-kind application that allows users to watch live and recorded TV shows and movies, listen to music, view slideshow photos, and play DVDs.

Additionally, Windows XP Media Center Edition comes with enhanced features:

  • Enhanced transition effects for Windows Movie Maker exclusively for this edition
  • Microsoft Plus! Features for this edition, such as:
    • Themes
    • Dancers
    • Party Time! (a feature that plays music and videos in a specified theme)
    • CD/DVD Burning Support
Windows XP Tablet PC Edition

Designed for convertible laptops, it supports handwritten notes and portrait-oriented screens. Windows XP Tablet PC Edition comes with Windows Journal, Sticky Notes, and more.

Windows XP Professional x64 Edition

Powered by the Windows Server 2003 architecture, Windows XP Professional x64 Edition packs the punch that is suitable for intensive computing tasks. It supports up to two physical CPUs, faster encoding for audio and video, and much more.

Windows XP Edition N and KN

For South Korea and the European Union countries. It functions the same as the original Windows XP, but without Windows Media Technologies to comply with the European Commission and the Korean Fair Trade Commission. Instead, Microsoft encourages the users within these countries to download third-party media players and instant messaging apps.

A special edition of Windows XP that is sold only to South Korea, called Windows XP Home Edition and Professional K, is also made available for purchase and comes with links to third-party media player and instant messaging applications.


Windows Vista and Windows 7

Windows Vista was released in January 2007. Some major changes to this operating system include Windows Aero, a graphical user interfacae that resembles the "window glasses," Windows Sidebar, a gadget software with clocks, calendars, etc., and Windows Search, which allows users to quickly search for documents/apps without having to look at all programs and/or documents. Internet Explorer 7 was also a major release of the browser; it includes the RSS platform, the Quick Tabs feature, and a steamlined user interface. Windows Media Center was included in Windows Vista Home Premium and Ultimate editions, excluding Home Basic and Business.

Windows Vista comes with four client editions:

Windows Vista Home Basic

Designed for basic computing needs, such as web surfing and checking emails. Windows Vista Home Basic comes with:

  • Windows Internet Explorer 7
  • Windows Defender (an anti-virus software)
  • Windows Media Player 11
  • Windows Mail
  • Windows Search
  • Parental Controls

Among other features in this edition.

Windows Vista Home Premium

The most preferred version of Windows for home desktops and mobile PCs. Windows Vista Home Premium comes with:

  • Fully-integrated Windows Aero user graphic interface
  • Premium games
  • Windows Media Center
  • Touchscreen support

Among other features in this edition.

Windows Vista Business

Helping your business start, grow, and thrive, Windows Vista Business comes with all features found in Home Basic PLUS:

  • The ability to join the Windows Server domain
  • Windows Fax and Scan
  • Remote Desktop
  • Internet Information Services

Among other features in this edition.

Windows Vista Ultimate

Marketed as "the most complete edition of Windows," Windows Vista Ultimate comes with all features found in Home Basic, Home Premium, and Business, PLUS the optional Windows Ultimate Extras, which include:

  • DreamScene: the ability to use your favorite video as a desktop background
  • Language packs for Windows multilingual interface
  • BitLocker: removes the complexity to use this exclusive data security feature.
  • More premium games

Two years after Vista was released, listening from criticisms and complaints by users, Microsoft released Windows 7 in October 22nd, 2009. It is rather an "improved version of Vista" and comes with several new features, such as the revamped taskbar for easy accessibility, more themes and sound schemes, and so much more.

Windows 7 comes in 4 consumer editions:

Windows 7 Starter

Designed for netbooks, Windows 7 Starter is suitable for users who want something simple with their basic home computing needs, such as the ability to surf the web, listen to some music, or check emails. It comes with:

  • Internet Explorer 8
  • Windows Snap (allowing a window to snap to one of the corners so that the other window can do the same)
  • Windows Media Player 12
  • Snipping Tool
  • Windows Search

Among other features in this edition.

Windows 7 Home Premium

The best preferred version of Windows. Windows 7 Home Premium comes with features found in Windows 7 Starter, plus:

  • Fully-integrated Windows Aero user graphic interface
  • Windows Media Center
  • Multi-touch support

Among other features in this edition.

Windows 7 Professional

Designed for business and power users, Windows 7 Professional comes with all features found in Starter and Home Premium, plus:

  • Windows XP Mode, a virtual machine package containing the pre-installed copy of Windows XP Professional, Service Pack 3, as a guest operating system. Windows XP Mode was made available to download free of charge, for users of Windows 7 Professional, Enterprise, and Ultimate.
  • The ability to join the Windows Server domain
  • Presentation Mode
  • Domain Join: the ability to connect to the company networks easily and more securely

Among other features in this edition.

Windows 7 Ultimate

Windows 7 Ultimate comes with all features found in Starter, Home Premium, and Professional combined together, PLUS:

  • The ability to help protect data on your PC with BitLocker
  • Multilingual Support through the Windows Language Packs.

Among other features in this edition.

Windows 8

Windows 8 is another major release from Microsoft that was made available on October 26th, 2012. Major changes to this operating system include replacing the traditional start menu with the so-called "Start" screen. The "Start" screen is optimized for touchscreen PCs, but it was criticized for making it confused to use, especially when it comes to keyboard and mouse. Another key feature found in this operating system is the revamped lock screen, now showing a background picture and a clock with the date on the left-corner of the screen; upon clicking the screen, swiping, or pressing a space bar, the secondary screen shows you the account(s) that may prompt you to enter your PIN number, your password, or even the picture password, which allows users to swipe anything that is in the picture to log in.

An update to Windows 8, named Windows 8.1, was released almost one year after the release of Windows 8. It addresses many concerns and criticisms found in Windows 8 by bringing back the visible start button. Another further update, called the April 2014 Windows 8.1 Update, brings keyboard and mouse closer together by showing the desktop by default when mouse is in use, as well as adding the search button and the shutdown options to the "Start" screen.

Windows 8 comes in 3 consumer editions:

Windows 8 Core

Windows 8 Core is designed for home computing needs. It comes with:

  • Internet Explorer 10
  • Windows Store (allows users to download their favorite apps)
  • Mail, Calendar, Photo, and SkyDrive (now known as OneDrive)
  • Multi-touch support

Among other features in this edition.

Windows 8 Pro

Windows 8 Pro comes with:

  • Data protection with BitLocker
  • Remote Desktop
  • Domain Join that connects to your corporate network.
  • Hyper-V Virtual Machines

Among other features in this edition.

Additionally, Windows 8 can be upgraded to the Windows 8 Pro Pack, which comes with Windows Media Center alongside all new features found in Windows 8.

Windows RT

Available only pre-installed on ARM-powered PCs, such as tablet PCs, Windows RT includes touch-optimized Microsoft Office 2013 RT for Home & Student. Apps can be downloaded only from the Windows Store due to limitations. Windows Media Player is not included.
