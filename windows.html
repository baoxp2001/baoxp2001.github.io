Windows 95

Windows 95 has been the well-known operating system launched in 1995. Some major features introduced to this operating system include a redesigned shell interface, the first Start menu, Plug-And-Play support, and "The Microsoft Sound" as a startup sound, created by Brian Eno.

Windows 95 at the Microsoft Visitor Center in Redmond, Washington.

Windows NT Workstation Version 4.0

Designed for business users, Windows NT Workstation Version 4.0 is suitable for intensive computing tasks and servers. It comes with the Windows 95 interface integrated into the Windows NT architecture, making this operating system more stable and easy to use. Windows NT 4.0 comes in multiple editions: Workstation, Server, and Embedded. It supports Alpha, MIPS, Intel, or PowerPC CPUs.

Windows 98

Released in 1998, Windows 98 is an improved version of the Windows 9x family. It comes with new features such as an improved user interface, shell enhancements, web integrations, and a robust USB support. Windows 98 can also support TV tuners through WebTV. Windows 98 Second Edition was released one year later, in 1999; it comes with improved USB support, improved audio and modem support, and an introduction to Internet Explorer 5.

Windows 98 also comes with the dedicated ACPI support which allows your computer to either hibernate or go into the stand-by mode, as well as the multi-monitor support, which uses up to 9 monitors from one PC.

Windows 2000 Professional

Built on Windows NT Technology, Windows 2000 Professional claims to be one of the most successful operating systems to date. It comes with an improved Plug-And-Play feature with full ACPI and Windows Driver Model support, usability enhancements, and mulit-language support. It comes with 3 editions: Professional, Server Family, and Advanced Server.

Windows Millennium Edition

Proven to be the last Windows 9x version, Windows Millennium Edition is suitable for home computing needs. It comes with improved boot times, Universal Plug-And-Play, Internet-based games, System Restore (which restores to the previous system from what was recently changed), and Automatic Updates. It also has enhanced multimedia features, such as...

  • Windows Media Player 7
    • Allows users to watch videos, listen to their favorite music, or make their own CDs!
  • Windows Movie Maker
    • Designed to edit videos together with music, videos, and titles!
  • Windows DVD Player
    • Plays DVD videos! Does not require a dedicated decoder for DVD playback.

Windows XP

Windows XP is the most popular operating system created by Microsoft. Launched on October 25th, 2001, XP unites both business and home users thanks to the power of Windows NT technology. It provides Wi-Fi and Bluetooth support for better connectivity. The Start menu has been revamped, now showing frequently used applications, recently opened documents, and so much more. The "Switch Users" feature allows computer users to switch to the other user without having to close all applications and log out.

Windows XP comes with two client editions:

  • Home Edition
    • Designed for home computing needs
  • Professional
    • Comes with multiple features not found in Home Edition, such as:
      • Remote Desktop, which connects the user's PC to operate by another user via the Internet
      • Offline Files and Folders
      • Group Policies
      • Encrypting File System

Additionally, Windows XP comes in several editions:

  • Media Center Edition
    • Allows users to watch live and recorded TV, view pictures, and listen to music with Windows Media Center.
  • Tablet PC Edition
    • Designed for convertible laptops, it supports handwritten notes and portrait-oriented screens. Windows XP Tablet PC Edition comes with Windows Journal, Sticky Notes, and more.
  • Professional x64 Edition
    • Powered by the Windows Server 2003 architecture, Windows XP Professional x64 Edition packs the punch that is suitable for intensive computing tasks. It supports up to two physical CPUs, faster encoding for audio and video, and much more.
  • Edition N and KN
    • For South Korea and the European Union countries. It functions the same as the original Windows XP, but without Windows Media Technologies. Instead, Microsoft encourages the users within these countries to download third-party media players and instant messaging apps.

Windows Vista and Windows 7

Windows Vista was released in January 2007. Some major changes to this operating system include Windows Aero, a graphical user interfacae that resembles the "window glasses," Windows Sidebar, a gadget software with clocks, calendars, etc., and Windows Search, which allows users to quickly search for documents/apps without having to look at all programs and/or documents. Internet Explorer 7 was also a major release of the browser; it includes the RSS platform, the Quick Tabs feature, and a steamlined user interface. Windows Media Center was included in Windows Vista Home Premium and Ultimate editions, excluding Home Basic and Business.

Windows Vista comes with four client editions:

  • Home Basic
    • Designed for basic computing needs, such as web surfing and checking emails.
  • Home Premium
    • Marketed as "the most preferred version of Windows," Home Premium comes with the fully-integrated Windows Aero themes, premium games, Windows Media Center, and touchscreen support.
  • Business
    • Windows Vista Business comes with Windows Fax and Scan, Remote Desktop, Internet Information Services, and more, so that your business can thrive and grow.
  • Ultimate
    • The complete version of Windows. Ultimate comes with all features found in Home Basic, Home Premium, and Business, plus BitLocker, mulit-language support, and the so-called "Windows Ultimate Extras," the optional features that include DreamScene, more premium games, and so much more!

Two years after Vista was released, listening from criticisms and complaints by users, Microsoft released Windows 7 in October 22nd, 2009. It is rather an "improved version of Vista" and comes with several new features, such as the revamped taskbar for easy accessibility, more themes and sound schemes, and so much more.

Windows 7 comes in 4 consumer editions:

  • Starter
    • Designed for netbooks, Windows 7 Starter is suitable for users with basic home computing needs, such as the ability to surf the web, check emails, or listen to their favorite music.
  • Home Premium
    • The most preferred version of Windows. Home Premium comes with Internet Explorer 8, Windows Media Player 12, the revamped Windows Media Center for live TV content, and multi-touch support.
  • Professional
    • Includes all features found in Home Premium, plus the ability to run older applications through Windows XP Mode, a virtual machine package containing the pre-installed copy of Windows XP Professional, Service Pack 3, as a guest operating system. Windows XP Mode was made available to download free of charge, for users of Windows 7 Professional, Enterprise, and Ultimate.
  • Ultimate
      • A complete version of Windows. Ultimate comes with all features found in Home Premium and Professional, plus BitLocker and mulit-language support.
